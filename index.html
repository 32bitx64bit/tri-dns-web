How to configure clients.

Rethink DNS

For DOT

Navigate to the DNS settings, click on "Other DNS"

Tap on DoT, then tap the ( + ) button.

[Optional] Set resolver name to "tri-dns"

Add "tls://tri-dns.net" to "Resolver URL"

For DOH (Unrecommended protocol)

Navigate to the DNS settings, click on "Other DNS"

Tap on DoH

[Optional] Set resolver name to "tri-dns"

Add "tri-dns.net/dns-query" to "Resolver URL"

Portmaster

For DOT

Navigate to the settings menu, and locate "DNS Servers"

Click "Add" and use this template "dot://(IP-HERE)?verify=(MATCHING-URL-LOCATION) "

[Optional] Add Ipv6 as well " dot://([iphere])?verify=(MATCHING-URL-LOCATION) "

For DOH (Unrecommended protocol)

Navigate to the settings menu, and locate "DNS Servers"

Click "Add" and use this template "https://(IP-HERE)?verify=(MATCHING-URL-LOCATION) "

[Optional*] Add Ipv6 as well " https://([iphere])?verify=(MATCHING-URL-LOCATION) "

Now make sure the IP matches with the service location, if you mismatch it'll just fail to resolve.

For Ipv6 the [] are needed inorder for it to work. (ipv6 only)
