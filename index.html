How to configure clients.

Rethink DNS

For DOT

Navigate to the DNS settings, click on "Other DNS"

Tap on DoT, then tap the ( + ) button.

[Optional] Set resolver name to "tri-dns"

Add "tls://tri-dns.net" to "Resolver URL"

For DOH (Unrecommended protocol)

Navigate to the DNS settings, click on "Other DNS"

Tap on DoH

[Optional] Set resolver name to "tri-dns"

Add "https://tri-dns.net" to "Resolver URL"

Portmaster

For DOT

Navigate to the settings menu, and locate "DNS Servers"

Click "Add" and paste " dot://178.128.225.109?verify=tri-dns.net "

[Optional] Add Ipv6 as well " dot://[2604:a880:cad:d0::728b:9001]?verify=tri-dns.net "

For DOH (Unrecommended protocol)

Navigate to the settings menu, and locate "DNS Servers"

Click "Add" and paste " doh://178.128.225.109?verify=tri-dns.net "

[Optional] Add Ipv6 as well " doh://[2604:a880:cad:d0::728b:9001]?verify=tri-dns.net "
